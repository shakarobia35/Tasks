# Markdown Note

## 1.基本用法
*学习完知乎和RUNOOB.COM里有关markdown的部分知识后，我们可以很"容易"的依据文章的分级以及走向归纳出其基本用法。*
### 标题---用以形成全文的粗纲
1. 使用`# `进行标题级别划分
- `# `*一级标题*
-   `## ` *二级标题*
-   > 一共可以分成六级标题。另外，在typora中亦可使用快捷键Ctrl+1、Ctrl+2...Ctrl+6设置一级到六级标题。

2. 使用`====`、`====`进行标题划分
    **利用=====和-----只能划分出两级标题，具有局限性。**

​    

### 列表---用以形成全文的细纲
1. 无序列表
    操作方法:用`* `或`+ `或`- `。
2. 有序列表
    操作方法:例如，`1. `。
3. 嵌套列表
    在需要进行嵌套的行前添加`  `。
4. 任务列表
    `- [ ]`表示未完成，`- [x]`表示已完成。

### 提纲之下的文本调整
**基于文本**
1. 分隔线
    `***`或`* * *`或`- - -`或`---`
    具体表现为:
    >我说完了
    >***

2. 删除
    在需要画删除线的文本前后加上`~~`
    eg:~~我被删除了TAT~~
3. 下划线
    在需要画下划线的文本前后分别加上`<u>`和`</u>`
4. 字体
    `*相关文本*`加粗
    `**相关文本**`斜体
    `***相关文本***`粗斜体
> 中英文混合时，在强调符号前后加空格可以提高可读性。
>   

5. 脚注
    表示方式:`[^要注明的文本]...`
   这里有一个注释[^1]
   [^1]:是的就是这样
7. 引用
  + 区块内用列表
    > 区块中使用列表
    > 1. 第一项
    > 2. 第二项
    > + 第一项
    > + 第二项
    > + 第三项

  + 列表内用区块
    * 第一项
    > 我们不追求赢也不追求输
    > 不追求快乐更不追求哭
    > 我们上去邦邦就是两拳
    * 第二项

**基于段落**
  + 段落换行时，一般是两个或以上空格加回车键。
  + 段落间用空行进行分隔
  + 普通段落不用` `或制表符缩进，而用`  回车`或`<br>`
### 插入
1. 表格
   先用|分割表头，下行注明对齐方式。对齐方式有: `:---`左对齐、`---:`右对齐、`:---:`居中，(默认居中)
   |表头1|表头2|表头3|
   |---:|:---:|:---|
   |右对齐|居中|左对齐|
   |a|b|c|
2. 代码
   用两个`夹着写行内代码，用两个```夹着写代码块。另外，可以在第一个```后加上想要使用的语言。
   这是一个`代码`
   ```python
   a,b=map(int,input())
   a+=b
   print(a)
   ```
3. 超链接
   + 直接在<>中加上网址；
   + 使用[文字](网址)的方式把链接附在文字上。
4. 图片
   + 用![替代文字](图片路径)的方式书写。若想分享给他人，需要把路径改写成相对路径，把图片和文章一起打包；
   + 使用网页图床，如<http://imgse.com>,上传图片后选择md格式。
5. 公式
   + 用两个$夹着写行内公式，用两个$$夹着写行内公式
   + 在md里的公式要通过Latex语法书写
     例如，
     `$x \times x = x^2$` $x \times x = x^2$
     `$$\left( \sqrt{x}+\sqrt y \right) \times \left( \sqrt{x}+\sqrt y \right) =x-y$$`
     $$\left( \sqrt{x}+\sqrt y \right) \times \left( \sqrt{x}+\sqrt y \right) =x-y$$
     `$$V=\int_{\alpha}^{\beta} \pi f(x)^2 dx$$`$$V=\int_{\alpha}^{\beta} \pi f(x)^2 dx$$
     > latex 基础数学公式
     > + `\times`\times
     > + `\div`\div
     > + `^`^上标`_`_下标
     > + `\frac{分子}{分母}`\frac{分子}{分母}
     > + `\sqrt{}`平方根`\sqrt[n]{表达式}`n次根号
     > + `\int_{下限}{上限}`积分  eg:\int_{0}^{\infty}e^{-x}\,dx  (`\,`小间距；`\;`大间距)
### 底层原理
***编写-解析-渲染***
1. **编写**
   + 在markdown里，我们需要手动输入语法符号告诉编译器需要对文本进行何种改动。这样，我们就可以将带有`.md`后缀的纯文本文件结构化
2. **解析**
   +各种markdown编辑器(如typora、VS Code)或平台(GitHub等)内部集合成一个"解析器"。
   +保存`.md`文件后，解析器会读取文件，并按照一套预设的规则，将诸如`## `的指令翻译出来。
3. **渲染**
   + 解析完成后，计算机就得到了HTML代码。这时，Chrome、Firefox等浏览器利用其渲染引擎读取HTML、CSS、JavaScript代码，并把它转换成最终看到的页面。
