# Markdown Note

## 1.基本用法
*学习完知乎和RUNOOB.COM里有关markdown的部分知识后，我们可以很"容易"的依据文章的分级以及走向归纳出其基本用法。*
### 标题---用以形成全文的粗纲
1. 使用`# `进行标题级别划分
- `# `*一级标题*
-   `## ` *二级标题*
-   > 一共可以分成六级标题。另外，在typora中亦可使用快捷键Ctrl+1、Ctrl+2...Ctrl+6设置一级到六级标题。

2. 使用`====`、`====`进行标题划分
    **利用=====和-----只能划分出两级标题，具有局限性。**

​    

### 列表---用以形成全文的细纲
1. 无序列表
    操作方法:用`* `或`+ `或`- `。
2. 有序列表
    操作方法:例如，`1. `。
3. 嵌套列表
    在需要进行嵌套的行前添加`  `。
4. 任务列表
    `- [ ]`表示未完成，`- [x]`表示已完成。

### 提纲之下的文本调整
**基于文本**
1. 分隔线
    `***`或`* * *`或`- - -`或`---`
    具体表现为:
    >我说完了
    >***

2. 删除
    在需要画删除线的文本前后加上`~~`
    eg:~~我被删除了TAT~~
3. 下划线
    在需要画下划线的文本前后分别加上`<u>`和`</u>`
4. 脚注
    表示方式:`[^要注明的文本]...`
5. 字体
    `*相关文本*`加粗
    `**相关文本**`斜体
    `***相关文本***`粗斜体
> 中英文混合时，在强调符号前后加空格可以提高可读性。
>   

6. 引用
  + 区块内用列表
    > 区块中使用列表
    > 1. 第一项
    > 2. 第二项
    > + 第一项
    > + 第二项
    > + 第三项

  + 列表内用区块
    * 第一项
    > 我们不追求赢也不追求输
    > 不追求快乐更不追求哭
    > 我们上去邦邦就是两拳
    * 第二项

**基于段落**
  + 段落换行时，一般是两个或以上空格加回车键。
  + 段落间用空行进行分隔
  + 普通段落不用` `或制表符缩进，而用`  回车`或`<br>`
